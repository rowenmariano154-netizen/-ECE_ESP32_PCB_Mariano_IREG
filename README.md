# ECE_ESP32_PCB_Mariano_IREG

## Project Description

This project presents the design and development of a custom ESP32-based expansion board intended for embedded systems, Internet of Things (IoT), and hardware prototyping applications. The board was designed to provide a structured and user-friendly platform for accessing the ESP32's GPIO pins while supporting stable power distribution and peripheral connectivity.

The design incorporates a USB Type-C power interface, expansion headers, filtering components, and control buttons to enhance usability during programming and testing. Special attention was given to component placement and PCB routing to achieve a compact layout while maintaining electrical reliability.

Using EasyEDA for schematic design, PCB layout, and 3D visualization, the project demonstrates the complete PCB development process—from concept and circuit design to board modeling and verification. The resulting hardware serves as a practical platform for learning, experimentation, and the development of wireless embedded applications.

## Features

- **ESP32-WROOM-32 Integration**
  - Provides Wi-Fi and Bluetooth connectivity for IoT applications.

- **GPIO Expansion Headers**
  - Easy access to ESP32 pins for interfacing sensors and peripherals.

- **USB Type-C Power Input**
  - Modern and convenient power connection.

- **EN and BOOT Controls**
  - Dedicated buttons for resetting and programming the ESP32.

- **Power Filtering Circuit**
  - Capacitors improve voltage stability and reduce electrical noise.

- **Compact PCB Layout**
  - Efficient arrangement of components for prototyping and development.

- **3D PCB Verification**
  - Design validated through EasyEDA's 3D visualization tools.
